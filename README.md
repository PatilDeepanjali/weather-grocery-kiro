Project: Weather vs Grocery Demand Predictor

Short description: Correlate weather (temperature, precipitation) with grocery search trends and visualize insights in a live dashboard. Use Kiro to automate data fetch, merge, and dashboard updates.

Tech stack: Python (data fetch & processing), Node/Static HTML (dashboard), Chart.js, Kiro workflow (workflow.yaml), GitHub Pages for hosting.
